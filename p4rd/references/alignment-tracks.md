# Alignment Tracks

Use this file only for `align` mode.

## Track 1: Memory Intelligence

Create tasks that cover:

1. Trace capture model
- Persist execution trajectories with chaptered events.
- Include actor, tool, decision, timestamp, significance.

2. Compaction pipeline
- Add retention + summarization jobs for long sessions.
- Keep source citations to raw traces.

3. Structured recall engine
- Blend lexical + semantic + recency + significance scores.
- Add offline evaluation fixtures and regression budgets.

4. Context graph projection
- Build entity/action/edge model from trajectories.
- Expose query APIs for process-pattern retrieval.

## Track 2: Remediation Autonomy

Create tasks that cover:

1. Signal normalization
- Ingest failing gates, incidents, and operator friction into one queue.

2. Remediation planner
- Convert signals into deterministic remediation plans.
- Prioritize by severity, blast radius, recurrence.

3. Execution loop
- Permit auto-remediation only under policy gates.
- Require approvals for risky categories.

4. Outcome learning
- Track success/failure of remediation actions.
- Feed outcomes back into planner heuristics.

## Quality Expectations

Each task criteria must include:
- Example path proving success.
- Negative path proving failure detection.

Each track should end with at least one integration task that verifies end-to-end behavior.
